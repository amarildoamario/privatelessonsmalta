# Curriculum Web — Next.js CV

This project scaffolds a simple Next.js site to render your CV and export it to PDF (A4) via the browser Print dialog.

Quick start:

```bash
npm install
npm run dev
```

Open http://localhost:3000, click "Export PDF (A4)" and choose "Save as PDF" with A4 paper.
